para plotar a correlaÃ§Ã£o vamos criar uma funÃ§Ã£o bastante simples. Ela receberÃ¡ o argumento cor que serÃ¡ nossa matrix de correlaÃ§Ã£o, e o argumento threshold que dirÃ¡ qual valor queremos usar como âcorteâ para salientar a signficÃ¢ncia

correlation.plot <- function(cor, threshold, ...){
  #aqui dizemos que qualquer valor menor que o threshold serÃ¡ equivalente Ã  1 no plot
